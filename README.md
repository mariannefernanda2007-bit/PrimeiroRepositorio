# Meu primeiro repositório
Este é o meu primeiro projeto que enviei para o github.

## Como enviar projetos para o github
-precisa ter o **GIT** instalado e o **github dDestop**
- Abrir o **CMD** na pasta do prejeto e executar o comando:  `git init`
- Fazer o **commit** no github Desktop e depois clicar em `push` para enviar
    -Se é a primeira vez, presica clicar em `publish`
    -Lembrar de, preferencialmente, deixar o projeto como **public**


## Como baixar projetos 
Abrir o CMD e digitar git clone [link do repositorio]
